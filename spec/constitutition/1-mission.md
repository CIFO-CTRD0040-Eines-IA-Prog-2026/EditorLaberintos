# Mission
- Product:
"Editor de laberintos" is an application that allows defining 2D mazes and navigate them in 3D

- Objective
The user must be able to edit a maze defined by a regular 2D grid where each cell can be either a wall (not walkable) or floor (walkable). This maze can be saved to a file and restored. At the same time, the user can navigate through the maze in an adjacent 3D view.
